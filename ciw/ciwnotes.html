XHTML

XHTML is a version of HTML that incorporates strict syntax rules. XHTML has three flavors ( transitionl, strict and Frameset.) In order for a XHTML doctype to be validated you need 1. The doctype statement 2. Document structure tags are required. 3. One root element( sole of the parent element like "html") 4. All elements must be appropiately nested 5. Attribute vales must be enclosed with quatation marks.
